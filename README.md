# Segre (Free/Libre & Open Source Private RTC messaging Application)
## What is Segre?
Segre is a Private Real Time Communication messaging application written in Dart programming language with zero bloat and tracker. All messages are end to end encrypted.

The application is chosen to be packaged as native binaries rather than a Browser based application having various of reasons behind :-
 1. Browsers are tends to be bloated with low performance and use a lots of resources and open up a browser is not good for privacy as well if it controlled by a firm like Google or Microsoft.
 2. Browsers track user searches for their businesses. 
 3. Since Browsers are products, so they track users for sending ads and track IP addresses, searches, etc which can easily blow up your privacy and kills the whole purpose of private messaging.
